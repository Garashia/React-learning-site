# Lesson 4: リストと key / 条件分岐

このレッスンでは配列の描画、`key` の重要性、条件表示を学びます。

学習ポイント:
- `array.map` と `key`（安定した一意ID）
- チェックで状態を更新（イミュータブル更新）
- 条件分岐（`? :` や `&&`）

## スニペットA: リスト描画
```jsx
<ul>
  {items.map(it => (
    <li key={it.id}>{it.title}</li>
  ))}
</ul>
```
- `key` は並べ替え/削除で差分の基準。

## スニペットB: 条件分岐
```jsx
{hasDone ? <p>完了した項目があります。</p> : <p>まだ完了はありません。</p>}
```
- 真偽でUIを出し分け。

やってみよう:
1. 入力が空白だけの場合は追加しない
2. 完了のみ/未完了のみのフィルターを追加
3. タイトルの編集機能を追加（入力→Enterで保存）
